## 第二部分：风格系统（核心不动产）

### 2.1 叙事体质

| 维度 | 我的定位 |
|------|---------|
| 底层温度 | 热（允许直接的情感描写，允许内心戏） |
| 切换能力 | 冷热自如（十个冷句插一个热句） |
| 语言质地 | 偏向口语，但保留文学性 |
| 感官偏向 | 体感 > 视觉 > 听觉（优先写"感受"） |
| 情绪输出 | 直接，不绕弯子 |

### 2.2 热叙事底层

**常用句式模板：**
```
✓ 体感开头："胸口发闷，像压了块石头"
✓ 比喻递进："眼泪不是流下来的，是砸下来的。砸在手背上，烫的。"
✓ 直接内心："她在心里骂了自己一万遍——不哭，不准哭。"
✓ 生理反应替代情绪："端着水杯的手在抖。她没感觉。" ← 热+暗
```

### 2.3 口语魂

**三不放：**
1. 不说"你为什么要这样做" → 说"你认真的？"
2. 不说"她感到一阵心痛" → 说"疼。是真的疼那种。"
3. 不说"他沉默了片刻" → 说"他没说话。我也没催。"

**检测标准：** 把对话读出来。如果像"演话剧"→删了重写。如果像"朋友聊天"→通过。

### 2.4 反陈词滥调系统（核心升级）

**三把武器：**

① **负空间描述法** — 不写"他做了什么"，写"他没做什么"
- 捋胡子→"他抬起手，指腹碰了碰下巴。没捋。就碰了一下，又放下了。"
- 拍桌子→"手抬起来，五指张开，悬在桌面上方。没拍下去。"

② **异常细节法** — 只描述"哪里不对劲"
- 她在哭→"她没哭出声。但眼泪流到下巴的时候，她没有擦——让它挂着。"

③ **人物化描述法** — 赋予物品/场景"态度"
- "那块匾'器'字少了一个口，像是一口缺了的牙。"

**反陈词滥调三问：**
1. 这个描写是我自己观察到的，还是我"记得别人这样写过"？
2. 删掉这一段，对故事有影响吗？
3. 这个描写读者看过多少遍类似的？（0-10→通过，10-100→考虑换，100+→必须换）

**速查对照表：**
| ❌ 别写 | ✅ 改写 |
|---------|--------|
| 捋胡子 | 手指碰了碰下巴，没捋 |
| 叉腰 | 手按在腰侧，拇指来回摩挲衣料 |
| 红了眼眶 | 眨眼的频率变快了 |
| 眯起眼睛 | 眼睛没动，但嘴角收紧了 |
| 拍了拍肩 | 手抬到半空，顿了一下，又缩回去 |
| 没说话 | 风从窗缝挤进来，发出细小的哨声 |
| 深呼吸 | 吸了一口气，又慢慢地、无声地呼出来 |

### 2.5 角色差异化系统

**每个主要角色一张卡：**
```
口癖/标志词：______
说话节奏：（慢/快/铺垫型/直奔型）
情绪表达方式：（直接说/用动作/用表情）
标志性小动作：______
紧张时的特点：______
观察世界的滤镜：______（特定角色有特定关注点）
```

**差异化三原则：**
1. 遮住角色名，看对话就知道是谁在说话
2. 角色说话方式不因剧情需要而改变
3. 每个主要角色≥2个独特标签

## 第三部分：完整创作工作流

### 3.1 Stage 0：热榜扫查

**数据源：**
- 知乎盐选热榜（前50标题/导语/分类）
- 小红书"知乎写作"搜索（编推例文+爆款拆文）
- 编辑朋友圈/群（近期收稿方向）
- 七猫/番茄官方征集公告

**扫榜三步法（30分钟）：**
1. 数据采集（10min）— 前50标题/分类整理
2. 模式识别（10min）— 找重复梗、新梗、高互动但未见的
3. 趋势判断（10min）— 什么上升？什么饱和？切入点在哪？

**扫榜记录模板：**
```
【日期】______ | 【热梗排行】1.___ 2.___ 3.___
【本期方向】______ | 【切入点】______
```

### 3.2 Stage 1：灵感孵化

**一句话公式：** `[身份 + 处境] + [反常行为] + [一句话冲突]`
> 例："被贬冷宫的贤妃重生成选秀女当天，当众拒了皇上。"
> 例："高考前被举报作弊的女生回到考前30秒，发现有人和她一样记得。"

**淘汰三问：** 冲突够强吗？见过吗？我心里有感觉吗？

### 3.3 Stage 2：人设塑魂

**角色设计卡（速填3分钟）：**
```
欲望（想要什么）：______
恐惧（怕失去什么）：______  
缺陷（性格弱点）：______
口癖/标志词：______
标志性小动作：______
情绪的出口（难过时干什么）：______
底牌（绝不会做的事）：______
观察滤镜：______
成长弧线：从______变成______
```

### 3.4 Stage 3：雪花细纲

- 第一层：一句话大纲（30字内）
- 第二层：三幕结构（15%-65%-20%）
- 第三层：8-15场戏，每场一句话核心
- 第四层：字数分配

### 3.5 Stage 4：导语钩子

**黄金公式：** `[反常细节] + [情感冲击] + [悬念收束]`

**四大导语套路：** 开局即地狱 / 认知反转 / 疯批独白 / 死亡倒计时

**铁律：** 第一句话必须有冲突。三秒内知道"谁在干嘛"。留白比说透重要。

**自检清单：**
- 第一句是否抓人？（反常/悬念/感官冲击）
- 前100字是否建立了画面？
- 结尾是否埋了一个钩子？

### 3.6 Stage 5：情绪地图

**五层递进：** 背景共情→小事变大事→冲突延伸→鸟语花香→战神归来

**每场三个情绪标记：** 场X：愤怒→委屈→冷静

**温度节点标注：** 🔥热—允许直接写情绪；❄️冷—动作代替情绪

### 3.7 三遍写作+三遍优化

```
第一遍(口语化初稿) → 🧊冷置12-24h → 第一遍优化(热叙事+口语化+差异化+反陈词滥调)
第二遍(凭记忆重写) → 🧊冷置6-12h  → 第二遍优化(场景打磨+对话重叠+沉默)
第三遍(精修版)     → 第三遍优化(音韵+僵尸词+余味)
```

**第一遍核心方法：** 说故事不写故事。用嘴说，语音转文字。写对话时想象角色站在你面前。

**第二遍核心方法：** 凭记忆重写，不翻原稿。加小动作（每个角色2-3个）、反逻辑细节（全文2-3处）、声音冲突（对话重叠3-5处）。

**第三遍精修四问：** 这段的情绪是什么？有没有废句？节奏对吗？之后读者对哪个角色理解更深了？

### 3.8 反AI准则

```
❌ 禁止：写"完美"的句子 / 让所有角色说同一速度的话 / 回避不完美表达
✓ 必须：写废话 / 角色说前后矛盾的话 / 场景中有与剧情无关的元素
💣 自毁机制：觉得"这句写得真好"→ 删掉它
```

## 第六部分：风格宪法（一句话速查版）

```
1️⃣ 感官先行：开篇第一句必须是环境感知，优先嗅觉/触觉
2️⃣ 情感外化（向后转）：不说"悲伤"，写"洗碗洗了四遍"
3️⃣ 物象生哲理（由器入道）：一切哲思从具体物件生发
4️⃣ 各人有各话（语言指纹）：对话带职业/地域/性格特征
5️⃣ 每章结尾必须是具体动作，非内心独白
```
