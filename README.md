# 创世银河小说创作工作系统 v3.1

> 短篇/中篇创作专用版。用最理性的工程框架，承载最疯狂的个人灵感。

## 一句话定位

**短篇小说创作的操作系统。** 从"我的风格是什么"到"读者为什么记住我"，每一步可执行、可验证。

## 文档结构

| 卷 | 名称 | 核心内容 |
|----|------|---------|
| 第一卷 | 战略定位层 | 母题深潜 + 平台基因精准适配（知乎/豆瓣） |
| 第二卷 | 故事架构层 | 短篇情绪弧 + 微冲突张力 + 三线钩 + 中篇结构模板 |
| 第三卷 | ★ 风格锻造层 | 认知底色 + 语言系统锻造 + 铁律×5 + 温度×5 + 导师×9 + 混合公式 + 语言指纹四步法 + 反AI味12类 |
| 第四卷 | 角色铸造层 | 反套路人设 + 标志微动作 + 七维语音 + 对比手法 + 反派升维 + 配角功能位 |
| 第五卷 | 执行生产层 | 精简6 Phase：概念→人物→骨架→快写→精修→门禁 |
| 第六卷 | 叙事辨识度 | 套路反常规 + IP记忆点 + 三层风格保卫 + AI协作三阶段 |
| 第七卷 | 投稿与运营 | 知乎盐选 + 豆瓣阅读 + 平台决策树 |
| 第八卷 | 速查+行动清单 | 写前/写后速查 + 风格门禁 + 风格锻造清单 |
| 附录 | 长篇模块索引 / 外部项目来源 / 版本演化 | 长篇参考 + 溯源 |

## 适用场景

| 场景 | 使用路径 |
|------|---------|
| 知乎盐选短篇 | 第一卷定位→第三卷风格→第二卷结构→第五卷执行→第七卷发布 |
| 豆瓣阅读中短篇 | 同上，第二卷使用中篇模板，风格锻造深度加强 |
| 风格突破/升级 | 第三卷全卷深度执行 + 第八卷风格锻造行动清单 |
| 新手起步 | 第一卷母题深潜 → 第五卷走通第一个短篇 |
| AI协作写作 | 第六卷 AI协作三阶段框架 |
| 长篇创作（如需） | 附录A 索引 → 调取创世银河 v2.0 完整长篇模块 |
