# 创世银河小说创作工作系统 v3.0

> 短篇/中篇创作专用版。用最理性的工程框架，承载最疯狂的个人灵感。

## 一句话定位

**短篇小说创作的操作系统。** 从"我的风格是什么"到"读者为什么记住我"，每一步可执行、可验证。

## 系统架构（三层 + 八卷）

```
认知层：找到你的母题、审美内核、个人辨识度
    ↓
执行层：风格锻造 → 角色铸造 → 故事架构 → 快写快修
    ↓
交付层：投稿策略 → 平台适配 → 数据反馈 → 迭代进化
```

## 适用场景

| 场景 | 使用路径 |
|------|---------|
| 知乎盐选短篇 | 第七卷 → 第二卷（3-5节拍） → 第五卷精简流程 |
| 豆瓣阅读中短篇 | 第七卷 → 第二卷（中篇模板） → 第三卷深度锻造 |
| 风格突破/升级 | 第三卷全卷深度执行 + 风格锻造行动清单 |
| 新手起步 | 序章 → 第一卷母题深潜 → 第五卷走通第一个短篇 |
| AI协作写作 | 第六卷 AI协作三阶段框架 |
