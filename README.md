# 创世银河小说创作工作系统 v2.0

> 神经叙事时代的终极创作操作系统。用最理性的工程框架，承载最疯狂的个人灵感。

## 一句话定位

**长篇连载 + 短篇投稿，双轨道覆盖。** 从"我是谁"到"读者为什么付费"，每一步可执行、可验证、可商业化。

## 系统架构（六层双螺旋）

```
工艺螺旋（v6.0精髓）              商业螺旋（V3.0 Pro精髓）
    ↓                                  ↓
风格精度 · 质量门禁               市场定位 · 平台基因
工具链 · 反AI检测                情绪工程 · 数据校准
角色语音 · 冷热切换               爽点放大 · 防崩盘
    ↓                                  ↓
          交汇 → 创世银河引擎 ← 交汇
```

## 系统亮点

- **风格混合系统**: 9位大师 × 基座70%+修饰30% × 场景偏移双矩阵 × 禁止模式取并集
- **反AI味检测**: 12类200+短语 × 高潮场景专项 × 风格模式感知阈值
- **五感签名**: 选定1-2种感官贯穿全部场景，用感官取代情绪词
- **四轮驱动引擎**: 实力60%+世界20%+矛盾+爽点，解决长篇崩盘
- **双轨道覆盖**: 长篇连载全六卷 + 短篇投稿精简6 Phase
