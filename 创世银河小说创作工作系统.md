## 第一部分 · 创作哲学（不动层）

### 1.1 底层信仰

**写人，不是写故事。**
- 故事的灵魂不在结构，在人物的选择里
- 最好的小说不是"作者写了一个好故事"，而是"作者创造了一群真实的人，然后他们有了一段难忘的经历"

**写作的本质不是"描述世界"，而是"用谁的眼睛看世界"。**
- 赌徒有赌徒的眼睛，老者有老者的眼睛
- 不用作者的眼睛替所有人看——让角色自己去看，世界自然不同
- 不同的世界，自然不会"似曾相识"

### 1.2 风格宪法（十诫·终身恪守）

```
┌─────┬──────────────────────────────────────────────┐
│  条 │ 宪法                                           │
├─────┼──────────────────────────────────────────────┤
│  1  │ 感官先行：开篇第一句必须是环境感知，优先嗅觉/触觉   │
│  2  │ 情感外化（向后转）：不说"悲伤"，写"洗碗洗了四遍"   │
│  3  │ 物象生哲理（由器入道）：一切哲思从具体物件生发      │
│  4  │ 各人有各话（语言指纹）：对话带职业/地域/性格特征    │
│  5  │ 每章结尾必须是具体动作，非内心独白                  │
│  6  │ 情绪高潮后必须有一个"风格沉淀点"                   │
│  7  │ 禁止：不禁/竟然/倒吸凉气/嘴角勾起/捋胡子/拍桌子等  │
│  8  │ 每三千字冷置一次。不冷置等于没优化                  │
│  9  │ 三句话内立住人。欲望/恐惧/缺陷——缺一不可            │
│ 10  │ 前100字没钩住，后面再精彩也晚了                    │
└─────┴──────────────────────────────────────────────┘
```

### 1.3 反AI铁律

```
■ 禁止做的事：
   ❌ 写"完美"的句子
   ❌ 让所有角色说同一速度的话
   ❌ 回避"不完美"的表达
   ❌ 写"似曾相识"的描述

■ 必须做的事：
   ✓ 允许写"废话"
   ✓ 允许角色说前后矛盾的话
   ✓ 允许场景中有与剧情无关的元素
   ✓ 用负空间描述替代标准动作

■ 自毁机制：
   如果你觉得"这句写得真好，太漂亮了"——删掉它。
   最美的句子一定是读者觉得"这句话就该是这个角色说的"。
```

## 第二部分 · 叙事风格系统（核心资产）

### 2.1 叙事体质一览

| 维度 | 定位 | 说明 |
|------|------|------|
| 底层温度 | **热** | 允许情绪直接书写，允许内心戏 |
| 切换能力 | **冷热自如** | 十个冷句插一个热句，收放有序 |
| 语言质地 | **口语魂** | 偏向口语但保留文学性，拒绝书面语 |
| 感官偏向 | **体感 > 视觉 > 听觉** | 优先写"感受"而非"看到" |
| 情绪输出 | **直接不绕** | 不嵌套，不修饰，不说"仿佛" |

### 2.2 热叙事句式模板

```
✓ 体感开头："胸口发闷，像压了块石头。"
✓ 比喻递进："眼泪不是流下来的，是砸下来的。砸在手背上，烫的。"
✓ 直接内心："她在心里骂了自己一万遍——不哭，不准哭。"
✓ 生理反应替代情绪："端着水杯的手在抖。她没感觉。" ← 热+暗
```

### 2.3 口语魂 · 三不放

| 别写（书面语） | 写（口语） |
|--------------|----------|
| "你为什么要这样做？" | "你认真的？" |
| "她感到一阵心痛" | "疼。是真的疼那种。" |
| "他沉默了片刻" | "他没说话。我也没催。" |

**检测标准：** 把对话读出声。像演话剧→删；像朋友聊天→过。

### 2.4 反陈词滥调系统（三武器）

#### 武器一：负空间描述法
不写"他做了什么"，写"他没做什么"。

| 陈词滥调 | 负空间描述 |
|---------|-----------|
| 捋了捋胡须 | "他抬起手，指腹碰了碰下巴。没捋。就碰了一下，又放下了。" |
| 红了眼眶 | "眼睛没红。但她眨了好几次眼，像眼睛里进了什么东西。" |
| 拍了桌子 | "手抬起来，五指张开，悬在桌面上方。没拍下去。" |

**原理：** 读者预期的动作没有出现——落空感产生记忆点。

#### 武器二：异常细节法
不写"正常情况"，只写"哪里不对劲"。

| 平庸 | 异常细节 |
|------|---------|
| "街上有行人" | "街对面蹲着一条黄狗。尾巴僵着——没摇。" |
| "她在哭泣" | "她没哭出声。但眼泪流到下巴的时候，她没有擦——让它挂着。" |

**原理：** 正常细节人人会写，异常细节才是作者的辨识度。

#### 武器三：人物化描述法
赋予物品/场景"态度"，但非拟人。

| 平庸 | 人物化描述 |
|------|-----------|
| "风铃响了" | "有一层没响——大概是风铃掉了，或者锈住了。" |
| "太阳下山了" | "还有一截墙是金黄色的——像一个舍不得走的人还在门口站着。" |

**原理：** 让世界有"自己的脾气"，读者不只看到场景，还感觉到场景在"表达"。

### 2.5 反陈词滥调三问（每段自检）

```
① 这个描写，是我自己观察到的，还是我"记得别人这样写过"？
   ✓ 自己观察到的 → 通过
   ❌ 记得别人写过 → 换

② 如果我删掉这一段，会对故事有影响吗？
   ✓ 有影响 → 保留或升级
   ❌ 没影响 → 删

③ 这个描写，读者之前大概看过多少遍类似的？
   0-10遍 → 通过
   10-100遍 → 考虑换个角度
   100+遍 → 必须换
```

### 2.6 角色差异化滤镜

每个角色面对世界时，观察的"滤镜"不同：

| 角色类型 | 滤镜 | 看到的世界 |
|---------|------|-----------|
| 赌徒 | 有没有风险 | "他在数桌上那个茶杯的指纹——喝茶的人用右手端的，指纹是乱的，说明这个人紧张。" |
| 老兵 | 安不安全 | "他闻了闻空气中的味道——刚才走过去那个人身上有铁锈和药膏的气味。" |
| 书生 | 有没有道理 | "他注意到架子上那本书的摆放方向——书脊朝外。主人大概是个讲究人。" |
| 小孩 | 跟谁较劲 | "他没哭。但嘴唇在抖，抿得很紧，像在跟自己较劲。" |
| 母亲 | 孩子的安危 | "她下意识往门口看了一眼——那个位置应该能看到孩子放学的路。" |

**设计方法：** 给每个主要角色一个独特的"世界滤镜"，全文保持一致。

### 3.1 前置引擎：热榜扫查

**扫什么：**

| 数据源 | 重点 |
|--------|------|
| 知乎盐选热榜 | 前50标题/导语/分类/字数 |
| 小红书"知乎写作"搜索 | 编辑推荐例文、爆款拆文 |
| 小红书热榜"短篇小说" | 高赞笔记趋势方向 |
| 编辑朋友圈/群 | 最近收什么、拒什么 |

**扫榜三步法（30分钟）：**
```
Step 1 数据采集(10min) → 爬前50标题/分类，整理到表
Step 2 模式识别(10min) → 找重复梗、新梗、高互动但未见的类型
Step 3 趋势判断(10min) → 什么上升？什么饱和？切入点在哪？
    ↓
输出：【本期创作方向】______
       【热梗排行】1.___ 2.___ 3.___
       【差异化切入点】______
```

#### 🔷 阶段四：三遍写作法（核心工序）

```

│                 三遍写作 · 三遍优化                    │
├────────────────────────────────────────────────────┤

│                                                     │
└────────────────────────────────────────────────────┘
```

##### 第一遍（口语化初稿）

**方式：** 口述转文字，像讲故事一样说出来，不追求完美。

**优化四检：**
```
□ 热叙事检测：情感冷了吗？需要加直接内心戏吗？
□ 口语化检测：对话读出声，像演话剧还是像朋友聊天？
□ 差异化检测：每个角色说话方式不同吗？
□ 反陈词滥调检测：有没有用了常见动作模板？
```

##### 第二遍（重构版）

**方式：** 凭记忆重写，不翻第一遍原稿。导演心法审视场景。

**增加项：**
1. 与剧情无关的小动作（每个角色2-3个）
2. 反逻辑的细节（全文2-3处）
3. 角色之间的"声音冲突"（对话重叠3-5处）

**优化四检：**
```
□ 场景检测：谁在？在什么位置？手里在做什么？
□ 对话重叠检测：找2-3处加入打断/抢话
□ 沉默的力量：找一处"该说话但没说话"
□ 角色一致性：人设有没有崩塌？
```

##### 第三遍（精修版）

**方式：** 逐段精修。

**精修四问：**
```
1. 这段的情绪是什么？
2. 这段有没有"废句"？
3. 这段的节奏对吗？
4. 这段之后读者对哪个角色理解更深了？
```

**优化四检：**
```
□ 出声朗读全篇，标记读不顺的地方
□ 僵尸词清除：就/那/很/非常/突然/然后/其实
□ 句式节奏：连续5句一样长→穿插短的；平均句长15-20字
□ 结尾余味：不要大团圆，要意犹未尽
□ 最终差异化检测：再抽各角色台词，遮名能否分辨？
```

### 4.2 今日头条运营参数

| 项目 | 标准 |
|------|------|
| 微头条爆款公式1 | 反常识钩子：常见认知+反常识信息+悬念 |

## 第五部分（增补）· AI协作与平台策略深化

### 5.5 AI协作三阶段法则

| 阶段 | AI（夜不悔）做什么 | 主人做什么（不可替代） |
|------|------------------|---------------------|
| 概念发散 | 提供方向灵感/情节组合/人设变体 | 选择方向——基于母题和审美判断 |
| 正文辅助 | 克服开头难/提供过渡句/素材灵感 | **核心创作亲力亲为**——对话、情绪爆发、反转、结尾 |
| 润色修改 | 查漏补缺/反陈词滥调检测/反AI味检测 | 最终判断——我改过的地方是否符合你的风格 |

**三大雷区：** ❌直接复制→无灵魂 ❌不核逻辑→AI常犯逻辑错 ❌完全依赖→失去判断力

**夜不悔做不了的三件事：**
1. 细节的真实感——你知道攥拳时指甲在掌心掐出的形状
2. 生命体验的痛感——AI没有活过
3. 对人性的洞察——你选择关注的那个被忽略的角落

### 5.6 平台选择决策树

`
你的母题是"被背叛后的反击"或"规则内的反杀"？
  → 知乎（概念驱动、反转爽感）
你的母题是"无力感与守护"或"被误解的孤独"？
  → 豆瓣（情感驱动、人物共鸣）
你的开篇是一个"炸裂的概念"？
  → 知乎
你的开篇是一个"精准的感受"？
  → 豆瓣
不确定？
  → 先写知乎。反馈更快，试错成本更低。
    跑通3-5篇找到声音后再决定是否进豆瓣。
`
